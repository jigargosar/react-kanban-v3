# Fun Project Ideas

Small side projects discussed for learning and portfolio.

## Shortlisted (6)

### 1. Pixel Art Editor

Canvas grid, color palette, draw pixels, export as PNG. Undo/redo with command stack pattern. Pure frontend, no backend needed. Most creatively satisfying — instant visual feedback.

### 2. Soundboard

Drag-and-drop audio clips onto a grid, trigger them with keyboard keys. Web Audio API + drag-and-drop. Playful and tactile.

### 3. Typing Speed Test

Random passages, real-time character matching, WPM calculation, accuracy tracking, personal best history. Addictive "one more round" loop.

### 4. Habit Tracker Heatmap

Define habits, check them off daily. Builds a GitHub-style contribution heatmap over time — darker square = more habits done that day.

Scope:
- Habits CRUD — add, rename, reorder, archive/delete, color-code
- Daily check-in — toggle habits for any date (today + past dates)
- Heatmap view — grid visualization, hover to see details
- Stats — current streak, longest streak, completion rate per habit

Data: one table of habits, one table of daily entries. Small enough for a weekend, deep enough to be interesting.

### 5. Bookmark Manager

Save links with auto-fetched metadata (title, favicon, description), tag them, fuzzy search. Full CRUD, search, tagging — shows real product building.

### 6. Pomodoro Timer with Stats

Track focus sessions, show daily/weekly charts, play ambient sounds during work blocks. State machine pattern, Notifications API.
